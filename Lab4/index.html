Preguntas

  1. Que diferencias y semejanzas hay entre Java y JavaScript?

    Semejanzas

    Son lenguajes Orientados a Objetos

    Pueden ser utilizados para la web

    Las sentencias se terminan con ; (punto y coma)

    Son multiplataforma

    Diferencias

    Java es un lenguaje de programacion y JavaScript es un lenguaje de scripting

    Las aplicaciones de Java corren en una maquina virtual, las de JavaScript en un navegador

    El codigo de Java debe ser compilado y el de JavaScript es un archivo de texto

    La sintaxis de JavaScript es menos estricta que la de Java
